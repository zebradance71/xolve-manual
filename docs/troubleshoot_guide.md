🛠️ トラブルシューティング（困ったときは）
[!IMPORTANT]
アプリの動作がおかしい、またはエラーが表示される場合は、まず以下の手順をご確認ください。

🆘 よくあるエラーコードと対処法
画面に英語の数字（エラーコード）が出た場合は、以下の手順で対応してください。

▼ 403 Forbidden（権限エラー）
意味: X（Twitter）側の設定が「読み取り専用」になっています。

解決: X Developer Portal で「Read and Write」権限を有効にし、連携をやり直してください。

▼ 401 Unauthorized（認証エラー）
意味: 連携キーが古いか、間違っています。

解決: Accounts 画面で連携情報を一度削除し、最新のキーを再設定してください。

▼ 429 Too Many Requests（回数制限）
意味: 短時間に操作をやりすぎました（X側の制限）。

解決: 15分〜24時間ほど時間を置いてから再度お試しください。

▼ MACHINE_MISMATCH（端末不一致）
意味: 別のPCで同じライセンスが使われています。

解決: 以前のPCで「ライセンス解除」を行うか、サポートまでお問い合わせください。

1. アプリが起動しない・画面が真っ白
原因: 一時的な読み込み失敗やメモリ不足。

対処:

アプリを完全に終了（タスクマネージャー等で確認）して再起動してください。

直らない場合は PC を再起動 してください。

2. ライセンス認証がうまくいかない
原因: コピー＆ペースト時のミス、またはネット接続。

対処:

ライセンスキーの前後に 「余計な空白スペース」 が入っていないか確認してください。

インターネットに接続されているか確認してください。

3. X（Twitter）への投稿・運用が失敗する
原因: 連携情報の不備、または X 側の仕様制限。

対処:

Accounts 画面で「連携テスト」が通るか確認してください。

画像サイズや動画形式が X の推奨（MP4等）か確認してください。

自動運用は アプリを起動している間のみ 動作します。

4. CSVが読み込めない
原因: ファイル形式の不一致。

対処:

配布している サンプルCSV と同じ形式（列順・ヘッダー名）か確認してください。

📩 どうしても解決しないときは
解決しない場合は、以下を添えてサポートまでご連絡ください。

症状（例：予約投稿が反映されない）

エラーメッセージ（画面の文言をコピー）

ご利用環境（Windows 11 / macOS 等）
